# 中国古代建筑结构健康监测大屏

基于 Flask + ECharts 的智慧大屏系统，可视化展示中国古代建筑（仅限四大类：民居、官府、皇宫、桥梁）的结构参数与健康状态。

## 功能特性
- 实时展示古建筑结构数据（柱径、斗拱密度、健康度）
- 按建筑类型统计与对比
- 响应式大屏设计，适配各种分辨率
- 模拟传感器数据实时更新
- 符合 4C 大赛主题要求

## 技术栈
- 后端：Flask
- 前端：ECharts + 原生 JavaScript
- 数据：Pandas
- 样式：CSS3 Flexbox

## 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 运行程序：`python app.py`
3. 打开浏览器访问：`http://127.0.0.1:5000`

## 数据说明
- 数据范围：仅限民居、官府、皇宫、桥梁（1911年前）
- 核心指标：柱径、斗拱密度、建造年代、结构健康度
- 数据来源：历史文献与公开数据库
